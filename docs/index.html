    func :: 自由 -> ビジネス

    目標

    1. 生活を再び安定化させる。

      現在可能な範囲から着実に社会に貢献し、今一度生活を安定化させていきます。

      • 再就職(A型就労支援への通所も想定)。
      • 予備費の確保(40万円目標)。
      • ウェブサービスの運用方法および関係各法を再整理。

    2. 自由で活発な表現の場を再構築する。

      安定化した生活を支えに、自由にかつ活発に表現活動を行えるウェブサービスを個人事業主として展開していきます。

      • 段階的なデプロイメント。
      • ユーザの利用状況および反応の定期的・定量的な調査。
      • 継続的な機能の追加・修正および継続可否の判断。

    3. 自由で活発な場から正しく健全なビジネスを生む。

      上記のサービスをより長く安定化させるため、着実かつ堅実にビジネスに発展させる道筋を模索していきます。

      • 関係各法を遵守する態勢の確保。
      • ユーザの権利を保護する態勢の確保。
      • 上記2項目を踏まえた健全で発展的なビジネスの構築
